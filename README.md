# vanilla-spa-sample

## Run

```
  $ git clone https://github.com/webkatu/vanilla-spa-sample
  $ cd vanilla-spa-sample/
  $ npm install
  $ node index.js
```

access to [http://localhost:4000](http://localhost:4000")
